Installation requirements

Visual Studio 2019 (>= 16.3.0)
Integrated SQL in Visual studio or standalone SQL Server instance
Asp.net core 3.0 SDK, download links below

    Windows: https://dotnet.microsoft.com/download/thank-you/dotnet-sdk-3.0.100-windows-x64-installer
    Mac: https://dotnet.microsoft.com/download/thank-you/dotnet-sdk-3.0.100-macos-x64-installer
    Linux: https://dotnet.microsoft.com/download/linux-package-manager/rhel7/sdk-3.0.100

Installation steps

1. Clone the repository from https://github.com/miroslav-tashonov/AdsSite
2. Build and clean the project, make sure asp.net core 3.0 sdk is installed in this step
3. Create Database via sql scripts located on ~\AdsSite\AdSite.Data\Scripts\ execute the scripts in this particular order inside your database management system
1.Create_Database.sql
2.Import_Script.sql
3.Import_Roles.sql
4.Import_InitialData.sql - you can edit and define your own admin credentials in this script 

default :
username: admin@email.com  password : Admin123!


4. Execute via IISExpress or host this project 
5. Login to the site with the credentials you have provided into  Import_InitialData.sql script.
